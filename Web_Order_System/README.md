# Web_Order_System
107大數據期末作業<br>
製作一個簡易點餐會員系統
# 環境
開發環境為`Visual Studio Code`<br>
伺服器`xampp`<br>
資料庫`phpmyadmin`
# 螢幕解析度
1920*1080
